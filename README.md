# Clicker-Empire-Game
## Description
コンピュータサイエンス学習サービスの**Recursion**のProject3で作成した「Clicker Empire Game」というシミュレーションゲームです。  
ゲームの仕様のみが与えられ、１から作成しました。  
仕事や投資、不動産等でお金を稼ぎ、一攫千金を目指しましょう！
